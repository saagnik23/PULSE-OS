# PULSE-OS

This is a premium, high-fidelity UI implementation for PULSE OS (originally based on the [Design Karyo OS UI](https://www.figma.com/design/hgw2xD7ErV4iIBhtwLERYQ/Design-Karyo-OS-UI) Figma design).

## Features
- **Modern Dashboard Layout**: Sleek sidebars, stats, and real-time status indicators.
- **AI Chat & AI Employees**: Dedicated workspaces for human-AI collaboration.
- **Automation Center**: Manage workflows, triggers, and scheduled jobs.
- **Business Brain**: Knowledge discovery, report generation, and business analysis dashboard.
- **Inventory & Analytics**: Rich visual representations of products, sales, and inventory levels.

## Running the Code

1. Install dependencies:
   ```bash
   npm i
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
